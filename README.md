## How to use
To run this project yoy need:
  1. Clone this repo:

    `git clone https://github.com/OSpyrydonov/todo-app.git`

  2. Open this repo on your IDE:

    `code todo-app`

  4. Istall depedencies:

    `npm i`
    
or
    
    `npm install`

  5. Start the project:

    `npm start`
